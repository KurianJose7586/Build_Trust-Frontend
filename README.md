# Build_Trust Marketplace - Frontend 🏗️

The frontend of the Build_Trust Marketplace, a platform connecting customers with verified construction specialists in India.

## 🚀 Tech Stack
- **Framework:** React 18 (Vite)
- **Routing:** React Router 7
- **Styling:** Custom Vanilla CSS (Optimized for performance)
- **AI Integration:** Seamless interface for Project Scoping AI

## 📦 Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/KurianJose7586/Build_Trust-Frontend.git
   cd Build_Trust-Frontend
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Configure Environment:**
   Create a `.env` file if you need to point to a specific backend URL:
   ```env
   VITE_API_URL=http://localhost:8005
   ```

4. **Run Development Server:**
   ```bash
   npm run dev
   ```

## 🛠️ Features
- **Smart Scoping:** Interactive AI agent to estimate project costs.
- **Worker Marketplace:** Search and filter verified specialists.
- **Multi-Role Dashboards:** Specialized views for Customers, Workers, and Admins.
- **Secure Payments:** Integrated Dodo Payments flow.

## 🌐 Deployment Strategy (Free)
- **Platform:** Vercel or Netlify.
- **Build Command:** `npm run build`
- **Output Directory:** `dist`
